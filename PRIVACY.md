# Privacy Policy

**PyFy Telemetry - F1 Performance Analysis (Beta)**

*Last updated: 2026-03-03*

---

## Summary

This application does not collect, transmit, or share any personal data. All data remains entirely local on your machine.

## Data Collection

This application does **not** collect:

- Personal information or identifiers
- Usage metrics or analytics
- Crash reports or diagnostics transmitted to external services
- Location data
- Device identifiers

## Network Activity

This application:

- Does **not** require an internet connection to operate
- Does **not** transmit any data to external servers
- Does **not** include an auto-updater or background services. Note: when installed via the Microsoft Store, the Store may apply updates through its own standard mechanisms.
- Operates exclusively on your local network to receive game telemetry via UDP

The application listens for UDP telemetry data sent by existing third-party games on your local machine (default: `127.0.0.1`). It can optionally forward this data to another local address for use with sim racing software. No data leaves your local network.
This application is not affiliated with or endorsed by any of the third-party games that it supports, nor their publishers/developers - see the LICENSE.md for more details. 

## Data Storage

All telemetry data (CSV files, raw captures, calibration data, charts, and configuration) is stored locally on your machine. No data is uploaded, synced, or shared by this application with any external service.

## Third-Party Services

This application does not integrate with or send data to any third-party services.

## Changes to This Policy

This privacy policy may be updated from time to time. Updated versions will be posted to this repository. Continued use of the application after changes are posted constitutes acceptance of the revised policy.

## Acceptance

By installing and using this application, you acknowledge that you have read and agree to this privacy policy. If you do not agree, please discontinue use and uninstall the application.
