Welcome

PyFy Telemetry captures real-time telemetry for F1 sim racing and turns it into practical, lap-by-lap and turn-by-turn performance insights you can actually use to find more lap time. Currently compatible with EA Sports F1 25*.

This guided tour walks you through the key features — analysis charts, session comparisons, external LLM-powered coaching insights, and key telemetry data.

Select a walkthrough from the menu, or browse the example data to get started.
